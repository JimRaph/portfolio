# Personal Portfolio Website

A minimalist, elegant, and responsive portfolio website built with **Next.js** and **Typescript** to show what I've been up to.

---

## Features

- Fast, responsive, and SEO-friendly
- Hero section with name, role, and call-to-action
- Projects section with live previews and GitHub links
- Skills section with technologies used
- Contact form (email integration ready)
- Clean, modern design with accessible layout
- Rate limiting applied

---

## Tech Stack

- Next.js
- Tailwind CSS
- React Icons
- Nodemailer
- Typescript
